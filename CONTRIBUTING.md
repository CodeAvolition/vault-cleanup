## Contributing

Im trying out [Conventional Commits](https://www.conventionalcommits.org/) with this repo:

| Prefix | When to Use | Version Bump |
|--------|-------------|--------------|
| `feat:` | New feature | Minor (1.x.0) |
| `fix:` | Bug fix | Patch (1.0.x) |
| `docs:` | Documentation only | No release |
| `chore:` | Maintenance | No release |

its nice for automatic releases withsemantic-release so it would be good to adhere to it
